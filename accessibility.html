המחויבות שלנו לנגישות

פארק המים יבנה רואה בנגישות ערך עליון. אנו פועלים להנגיש את אתר האינטרנט שלנו ואת מתקני הפארק לכלל הציבור — לרבות אנשים עם מוגבלויות פיזיות, חושיות, קוגניטיביות ושכליות, בהתאם להוראות חוק שוויון זכויות לאנשים עם מוגבלות, התשנ"ח-1998, ולתקנות שוויון זכויות לאנשים עם מוגבלות (התאמות נגישות לשירות), תשע"ג-2013.

רמת תאימות

אתר זה נבנה בהתאם להוראות תקן ישראלי ת"י 5568 ולקווי המנחה של WCAG 2.1 ברמת תאימות AA, המהווים כיום את הסטנדרט הבינלאומי המוביל לנגישות אתרי אינטרנט.

אמצעי הנגישות הזמינים באתר

בכל עמודי האתר זמין כפתור נגישות (אייקון בצד שמאל למטה), המספק את האפשרויות הבאות:

  • 📝 התאמת גודל גופן — הגדלה והקטנה של הטקסט (עד 200%)
  • 🎨 ניגודיות גבוהה — שחור על צהוב לראייה ירודה
  • 🔁 היפוך צבעים — לחוויית קריאה נוחה יותר
  • ⚫ שחור-לבן — הסרת צבעים להתמקדות בתוכן
  • 📖 פונט קריא — גופן ידידותי לדיסלקטים
  • 🔗 הדגשת קישורים — סימון בולט של כל הקישורים בעמוד
  • 🖱️ סמן עכבר גדול — סמן מוגדל ובולט יותר
  • ⏸️ עצירת אנימציות — לסובלים מהפרעות קוגניטיביות
  • ⌨️ הדגשת מיקוד מקלדת — מסגרת בולטת לאלמנט שבמיקוד
  • ↺ איפוס הגדרות — חזרה מהירה לתצוגת ברירת מחדל
  • ⏭️ דילוג לתוכן עיקרי — קישור בראש כל עמוד (Tab)
הגדרותיך נשמרות: כל הגדרת נגישות שתפעיל תישמר בדפדפן שלך ותחול אוטומטית בעמודים הבאים שתבקר בהם.

ההתאמות שבוצעו באתר

  • ✓ מבנה סמנטי תקני (HTML5) — שימוש בתגיות header, nav, main, section, footer
  • ✓ היררכיית כותרות הגיונית — H1 → H2 → H3 בכל עמוד
  • ✓ טקסט חלופי לכל התמונות (alt text) — מותאם להקראה ע"י קוראי מסך
  • ✓ תוויות (labels) לכל שדות הטופס
  • ✓ תוויות ARIA בכפתורים, חלונות דיאלוג ותפריטים
  • ✓ ניגודיות צבעים מינימלית 4.5:1 בכל הטקסטים
  • ✓ ניווט מלא במקלדת — ללא צורך בעכבר
  • ✓ סימני מיקוד ברורים (focus rings) לכל אלמנט אינטראקטיבי
  • ✓ הודעות שגיאה ברורות בטופס יצירת קשר, בעברית או באנגלית
  • ✓ תמיכה דו-לשונית — עברית (ראשי) + אנגלית, עם lang מתאים
  • ✓ RTL מלא — כיוון מימין לשמאל בכל הדפים
  • ✓ אזורי לחיצה גדולים — כפתורים מינימום 44×44 פיקסל
  • ✓ טקסט שניתן להגדלה ללא שבירת פריסה (עד 200%)
  • ✓ קישור "דלג לתוכן" בראש כל עמוד למשתמשי מקלדת/קוראי מסך
